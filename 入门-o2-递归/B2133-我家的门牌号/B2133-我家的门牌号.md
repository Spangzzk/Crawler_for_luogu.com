# 我家的门牌号
 题目描述
----

 我家住在一条短胡同里，这条胡同的门牌号从 111 开始顺序编号。


若其余各家的门牌号之和减去我家门牌号的两倍，恰好等于 nnn，求我家的门牌号及总共有多少家。数据保证有唯一解。


 输入格式
----

 一个正整数 nnn。n<100000n<100000n<100000。


 输出格式
----

 一行，包含两个正整数，分别是我家的门牌号及总共有多少家，中间用单个空格隔开。


  输入输出样例
------

 **输入 #1** 
 复制
  
```
100

```
 **输出 #1** 
 复制
  
```
12 16

```
